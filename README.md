# Rockserwis Podcaster

Simple podcast player for rockserwis.fm internet radio.

## Features

- playing podcasts :)
- favorited podcasts
- favorited episodes
- dark/light theme
- history of listened episodes
- tracking the progress of the listened episode
- sorting episodes based on date field
